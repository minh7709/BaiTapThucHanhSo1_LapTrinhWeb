## 📋 Yêu cầu hệ thống
Trước khi bắt đầu, hãy đảm bảo máy tính của bạn đã cài đặt:
- **Node.js**: Phiên bản 18.0 trở lên (Khuyến nghị bản LTS).
- **npm** (thường đi kèm với Node.js).

```Bash
npm install
```
## 🚀 Hướng dẫn cài đặt và khởi chạy

Thực hiện các bước sau để thiết lập dự án trên môi trường cục bộ:

### 1. Sao chép mã nguồn (Clone)
Mở terminal và chạy lệnh sau:
```bash
git clone https://github.com/minh7709/BaiTapThucHanhSo1_LapTrinhWeb.git
cd BaiTapThucHanhSo1
```
### 2. Chạy Dự án
```Bash
npm run dev
```
truy cập: http://localhost:5173/
